# KomuniCare - Barangay Health and Wellness Platform

A web-based barangay health platform connecting residents with Barangay Health Workers (BHWs) for health logging, appointments, tasks, rewards, and emergency SOS.


## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
